# Screenshots

Add images here and reference them from the top-level `README.md`.

Suggested shots (keep them ~1600px wide, PNG or WebP):

| File | What to capture |
|------|-----------------|
| `desktop.png` | Desktop with the CRT shader on: bar + terminal + wallpaper |
| `crt-detail.png` | A close crop showing scanlines / curvature / aberration |
| `border.png` | The swirling four-color active window border |
| `lockscreen.png` | The optional PS1 memory-card lock screen |

Capture on Omarchy with:

```bash
omarchy capture screenshot
```

A close crop (`crt-detail.png`) sells the CRT effect better than a full desktop
shot — the scanlines and curvature read best up close.

Then commit the files and update the image paths in `README.md`.
